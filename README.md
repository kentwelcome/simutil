# simutil
iOS Simulator Command Line Utility

#Usage
```
usage: simutil.py [-h] {list,create,delete,launch,install,uninstall} ...

Utility to cotrol iOS Simulator.

positional arguments:
  {list,create,delete,launch,install,uninstall}
                        Support subcommands
    list                Show the status of iOS Simulator.
    create              Create new device of iOS Simulator.
    delete              Delete new device of iOS Simulator.
    launch              Launch iOS Simulator with selected device.
    install             Install iOS application into selected device.
    uninstall           Uninstall iOS application from selected device.

optional arguments:
  -h, --help            show this help message and exit
```
